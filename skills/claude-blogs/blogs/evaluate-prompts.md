# Evaluate prompts in the developer console

**Source:** https://claude.com/blog/evaluate-prompts  
**Published:** July 9, 2024  
**Categories:** Product announcements, Claude Platform, Coding

## Overview

The Anthropic Console now includes prompt generation and evaluation features to streamline prompt development for AI applications.

## Generate Prompts

Built-in prompt generator using Claude 3.5 Sonnet. Describe your task (e.g., "triage customer support requests") and receive an automatically generated, high-quality prompt. Supports automatic generation of input variables for testing, or manual test case input.

## Generate a Test Suite

The Evaluate feature tests prompts against diverse real-world inputs within the Console—no more manual spreadsheet management.
- Add test cases manually, import from CSV, or auto-generate from Claude
- Granular control over variable generation requirements
- Execute all test cases simultaneously with a single click

## Evaluate and Iterate

- Create new prompt versions and re-run test suites for rapid improvement
- Side-by-side comparison of two or more prompt outputs
- Subject matter experts grade responses on a five-point scale to measure whether modifications improved quality

## Getting Started

Test case generation and output comparison available to all Anthropic Console users. Documentation available in official developer docs.
