# Skills for organizations, partners, the ecosystem

**Source:** https://claude.com/blog/organization-skills-and-directory  
**Published:** December 18, 2025  
**Categories:** Product announcements, Claude apps, Agents

## Overview

Anthropic has expanded its skills feature making it simpler for organizations to deploy, discover, and build custom capabilities. Includes centralized management tools for admins, a growing directory of partner-built skills, and an open standard enabling skills to work across different AI platforms.

## Organization-Wide Skill Management

Team and Enterprise plan administrators can now provision skills centrally through admin settings. Once enabled by admins, these skills become available to all users by default.

## Simplified Skill Creation and Discovery

- Users can describe desired workflows and have Claude assist in construction, or write instructions directly
- For intricate processes, upload skill folders or leverage the skill-creator tool
- Claude helps modify existing skills
- Updated previews display complete contents before activation

## Partner Skills Directory

A curated collection of partner-built skills is accessible at claude.com/connectors. Major contributors include Notion, Canva, Figma, and Atlassian.

- **Atlassian**: Skills enable Claude to understand Jira and Confluence workflows beyond basic connectivity
- **Canva**: Users can create multi-platform campaigns through single prompts
