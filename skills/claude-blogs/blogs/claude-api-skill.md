# Claude API skill now in CodeRabbit, JetBrains, Resolve AI, and Warp

**Source:** https://claude.com/blog/claude-api-skill  
**Published:** April 29, 2026  
**Categories:** Agents, Claude Code, Claude Enterprise

## Overview

Anthropic announced that CodeRabbit, JetBrains, Resolve AI, and Warp are bundling the `claude-api` skill. The skill helps developers write production-ready Claude API code in the tools where they already work.

## What the Skill Helps With

The `claude-api` skill captures implementation details for Claude API development, including agent patterns, model migration, prompt caching, context compaction, and Managed Agents onboarding.

Developers can ask Claude to:

- Improve prompt cache hit rates
- Add context compaction to an agent
- Upgrade code to the latest Claude model
- Build deep research agents with Claude Managed Agents

In Claude Code, the post mentions commands such as `/claude-api migrate` and `/claude-api managed-agents-onboard`.

## Partner Availability

The skill is available in Claude Code, CodeRabbit, JetBrains, Junie, Resolve AI, and Warp. Anthropic also points coding-agent builders to the open-source `anthropics/skills` repository and a bundling guide.
