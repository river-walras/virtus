# Product development in the agentic era

**Source:** https://claude.com/blog/product-development-in-the-agentic-era  
**Published:** April 29, 2026  
**Categories:** Agents, Claude Platform

## Overview

Jess Yan, product manager for Claude Managed Agents, explains how Claude, Claude Code, Claude Cowork, and Managed Agents change product-management work by letting PMs prototype, automate, and explore product ideas directly.

## Product Development Workflow

Yan uses Claude and Claude Cowork for open-ended research and discovery, then uses Claude Code to build custom agents on top of Managed Agents once the job to be done becomes clear.

Claude Code is used to prototype against pre-production API specs, test product abstractions, and uncover product or API issues earlier than document review alone would.

## Managed Agents Use Cases

Examples include:

- Adoption analytics agents that query internal databases and retain memory across runs
- Developer sentiment monitoring agents that use web search and parallel research
- Demo-building agents that customize templates with GitHub repos, branding assets, and event decks

Because Managed Agents run in the cloud, long-running tasks can continue while the PM works elsewhere.

## Product Craft

The post argues that agentic tools free product managers from repetitive operational work and create more room for customer discovery, engineering partnership, and product judgment.
