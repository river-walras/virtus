# Claude Managed Agents: get to production 10x faster

**Source:** https://claude.com/blog/claude-managed-agents  
**Published:** April 8, 2026  
**Categories:** Product announcements, Claude Platform, Agents

## Overview

Anthropic has launched Claude Managed Agents, a suite of composable APIs designed to accelerate the development and deployment of cloud-hosted agents. The platform eliminates infrastructure complexity, enabling teams to move from prototype to production in days.

## Key Features

**Production-Grade Infrastructure:** Handles secure sandboxing, authentication management, and tool execution. Teams no longer need to build these components independently.

**Long-Running Sessions:** Agents can operate autonomously for extended periods, maintaining state even through disconnections.

**Multi-Agent Coordination:** (Research preview) Allows agents to spawn and direct other agents, enabling parallel task execution.

**Governance and Security:** Built-in scoped permissions, identity management, and execution tracing.

## Performance Improvements

On structured file generation tasks, Managed Agents "improved outcome task success by up to 10 points over a standard prompting loop, with the largest gains on the hardest problems."

## Real-World Implementations

- **Notion**: Teams delegate work to Claude within their workspace for parallel task execution
- **Rakuten**: Enterprise agents deployed across departments integrated with Slack and Teams
- **Asana**: AI Teammates work alongside humans inside projects
- **Vibecode**: Customers move from prompts to deployed applications with dramatically accelerated timelines
- **Sentry**: Paired debugging agents with patch-writing capabilities, reducing bug-to-PR workflow from months to weeks

## Availability

Available in public beta on the Claude Platform.
