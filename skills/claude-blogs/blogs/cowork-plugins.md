# Customize Cowork with plugins

**Source:** https://claude.com/blog/cowork-plugins  
**Published:** January 30, 2026  
**Categories:** Product announcements, Claude Enterprise, Agents

## Overview

Anthropic has announced plugin support for Cowork, enabling users to customize Claude for specific roles and workflows. Plugins bundle skills, connectors, slash commands, and sub-agents together.

## Key Features

Plugins let you "tell Claude how you like work done, which tools and data to pull from, how to handle critical workflows." A sales plugin, for example, could connect Claude to a CRM and knowledge base while exposing commands for prospect research and follow-ups.

## Open-Source Plugin Collection (11 plugins)

- Productivity and task management
- Enterprise search and information discovery
- Sales enablement and deal preparation
- Financial analysis and modeling
- Data querying and visualization
- Legal document review and compliance
- Marketing content and campaign planning
- Customer support workflows
- Product management and roadmaps
- Biology research and literature analysis

## Availability

Plugin support available as research preview for all paid Claude users. Install plugins directly from Cowork, customize existing ones, or create new plugins using the Plugin Create tool.

Currently, plugins save locally to individual machines, with organization-wide sharing and private plugin marketplaces coming in subsequent releases.
