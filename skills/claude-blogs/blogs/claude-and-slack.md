# Claude and Slack

**Source:** https://claude.com/blog/claude-and-slack  
**Published:** October 1, 2025  
**Categories:** Product announcements, Claude Enterprise

## Overview

Two complementary integrations: add Claude directly to your Slack workspace, or connect Slack to Claude apps to search conversations and pull context during interactions.

## Using Claude in Slack

Three access methods:
- **Direct messages**: Private conversations with @Claude for research, writing, or analysis
- **AI assistant panel**: Access via icon in Slack's AI assistant header without interrupting channels
- **Thread participation**: Mention @Claude in any thread; Claude drafts responses based on context for review before sharing

Claude maintains access to web search and document analysis. Respects all Slack permissions—only accesses channels you can view. Responses in public threads are drafted privately first.

## Connecting Slack to Claude

When integrated, Claude searches workspace channels, DMs, and files for context:
- **Meeting preparation**: Retrieve relevant discussions and create briefings
- **Project coordination**: Gather status updates and identify blockers across channels
- **Onboarding support**: Help new employees understand projects via channel history
- **Documentation**: Formalize team discussions into formal records

## Security and Availability

- Respects organizational retention policies and security settings
- Workspace administrators control app approval
- Individual users authenticate with existing Claude accounts

**Claude in Slack**: Available via Slack Marketplace for paid plans.  
**Slack connector**: Requires Claude Team or Enterprise subscriptions. (Also available for Pro and Max subscribers as of January 2026.)
