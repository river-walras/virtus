# Eight trends defining how software gets built in 2026

**Source:** https://claude.com/blog/eight-trends-defining-how-software-gets-built-in-2026  
**Published:** January 21, 2026  
**Categories:** Agents, Claude Code

## Overview

The software development landscape is undergoing fundamental transformation as engineering teams shift from writing code to orchestrating AI agents.

## Key Finding: The Collaboration Model

While developers utilize AI in approximately 60% of their work, they can "fully delegate" only 0-20% of tasks. Effective AI adoption requires active human supervision, validation, and judgment.

## Real-World Applications

**Rakuten**: Deployed Claude Code on a complex task involving activation vector extraction in vLLM's 12.5-million-line codebase. Completed autonomous work in seven hours, achieving 99.9% numerical accuracy.

**TELUS**: Engineering teams created over 13,000 custom AI solutions while accelerating code shipping by 30%, representing over 500,000 hours of saved work.

**Zapier**: Achieved 89% AI adoption across their organization with 800+ internally deployed agents.

## Strategic Priorities for 2026

Organizations should focus on:
1. Mastering multi-agent coordination
2. Scaling human-agent oversight through AI-automated review
3. Extending agentic coding beyond engineering departments
4. Embedding security architecture from initial development stages

## The Path Forward

Human expertise now concentrates on architecture, system design, and strategic decisions—the highest-value activities—while AI handles implementation workflows including testing, debugging, and codebase navigation.
