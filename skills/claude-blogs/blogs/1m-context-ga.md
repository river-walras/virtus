# 1M Context is Now Generally Available for Opus 4.6 and Sonnet 4.6

**Source:** https://claude.com/blog/1m-context-ga  
**Published:** March 13, 2026  
**Categories:** Product announcements, Claude Platform, Agents

## Overview

Anthropic has made the full 1-million-token context window generally available for both Claude Opus 4.6 and Sonnet 4.6 at standard pricing with no premium surcharge.

## Key Announcements

**Unified Pricing Structure**
Standard rates now apply across the entire context window: $5/$25 per million tokens for Opus 4.6 and $3/$15 for Sonnet 4.6.

**Enhanced Capabilities**
- Full throughput limits accessible at every context length
- Media capacity expanded sixfold—up to 600 images or PDF pages per request
- Available on Claude Platform, Microsoft Foundry, and Google Cloud's Vertex AI
- Automatic activation without requiring beta headers

**Integration with Claude Code**
Max, Team, and Enterprise users gain access to 1M context with Opus 4.6, enabling fewer context compactions.

## Performance Metrics

Opus 4.6 achieves 78.3% accuracy on MRCR v2 benchmarks—the highest score among frontier models at full context length.
