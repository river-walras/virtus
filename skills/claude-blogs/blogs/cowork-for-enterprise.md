# Making Claude Cowork Ready for Enterprise

**Source:** https://claude.com/blog/cowork-for-enterprise  
**Published:** April 9, 2026  
**Categories:** Product announcements, Claude Enterprise, Business

Claude Cowork is now generally available on all paid plans. The platform handles tasks and project deliverables while keeping teams informed.

## New Enterprise Controls

**Role-based access controls** allow Enterprise admins to organize users into groups—either manually or via SCIM integration—and assign custom roles defining which Claude capabilities each group can access.

**Group spend limits** let organizations set per-team budgets from the admin console, providing cost predictability.

**Usage analytics** now display Claude Cowork activity in admin dashboards and via the Analytics API. Admins can track sessions, active users across date ranges, per-user activity, skill invocations, and engagement metrics (DAU/WAU/MAU).

**Expanded OpenTelemetry support** enables Claude Cowork to emit events for tool calls, file operations, skills used, and approval status. Events integrate with standard SIEM platforms like Splunk and Cribl.

**Zoom MCP connector** brings meeting intelligence directly into Cowork, delivering AI-generated summaries and action items alongside transcripts.

**Per-tool connector controls** let admins restrict actions within MCP connectors organization-wide—permitting reads while disabling writes, for example.

## Real-World Adoption

Organizations like Zapier, Jamf, and Airtree have deployed Claude Cowork for engineering bottleneck analysis, performance reviews, and board preparation workflows.
