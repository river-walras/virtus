# How Brex improves code quality and productivity with Claude Code

**Source:** https://claude.com/blog/how-brex-improves-code-quality-and-productivity-with-claude-code  
**Published:** October 30, 2025  
**Categories:** Enterprise AI, Claude Code, Coding

## Overview

Brex demonstrates how Claude Code transforms work across technical and non-technical roles, with productivity gains alongside fundamental shifts in how teams approach problem-solving.

## Breaking Traditional Barriers

Content designer Andy Reed now independently handles code changes and recently completed a multi-week design system project in days by integrating content guidelines across components. "It would have taken me probably weeks if not months doing it manually."

## Engineering Reimagined

The focus transitions from implementation details to architecture and vision, enabling teams to evaluate multiple approaches simultaneously. Claude Code serves as an "oracle" within Brex's complex monorepo structure, instantly answering questions about codebase functionality.

## Data Democratization

Brex built **Brex Explorer**, a text-to-SQL interface powered by Claude Code. Sales managers now query databases in plain language without SQL expertise. The team achieved approximately 50% adoption with targets for complete rollout.

## Emerging Best Practices

- **CLAUDE.md files** in major directories containing domain-specific context
- **Automated documentation** via CI/CD checks alerting teams to outdated information
- **Custom commands** pre-loading relevant context before execution
