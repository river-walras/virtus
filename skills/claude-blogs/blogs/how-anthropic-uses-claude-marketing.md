# How Anthropic's Growth Marketing team cut ad creation time from 30 minutes to 30 seconds with Claude Code

**Source:** https://claude.com/blog/how-anthropic-uses-claude-marketing  
**Published:** January 26, 2026  
**Categories:** Enterprise AI, Claude Enterprise

## Overview

Austin Lau, a growth marketer at Anthropic with no prior coding experience, built two automation systems in a week that reduced ad creation time from 30 minutes to 30 seconds.

## Solutions Built

### Figma Plugin for Creative Variations
Built in ~45 minutes. Users specify a frame, paste headline variations from a Google Sheet, and the plugin generates all permutations with one click. Saves ~30 minutes per large creative batch update.

### Google Ads Copy Generation Workflow
Uses a `/rsa` slash command that:
- Accepts campaign data, existing copy, and keywords
- References custom skills for brand tone, product accuracy, and Google Ads best practices
- Generates upload-ready CSV files after human refinement

## Key Takeaways for Non-Technical Users

- **Start small**: First project was a simple calculator
- **Explain clearly**: No coding knowledge required—describe challenges in plain language
- **Leverage existing resources**: Point Claude Code toward API documentation
- **Maintain human judgment**: All copy reviewed collaboratively with copywriting teams

## Broader Organizational Impact

- Influencer marketing saves 100+ hours monthly on script writing
- Customer marketing reduces case study creation from 2.5 hours to 30 minutes
- Digital marketing achieved 5x productivity increase year-over-year
- Product marketing saves 5-10 hours per launch
