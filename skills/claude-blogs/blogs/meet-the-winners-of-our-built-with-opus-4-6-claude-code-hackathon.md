# Meet the winners of our Built with Opus 4.6 Claude Code hackathon

**Source:** https://claude.com/blog/meet-the-winners-of-our-built-with-opus-4-6-claude-code-hackathon  
**Published:** April 20, 2026  
**Categories:** Claude Code

## Overview

Anthropic profiles winners from the Built with Opus 4.6 Claude Code hackathon. Four of five winners were not professional developers, and projects spanned housing, education, healthcare, infrastructure, and music.

## Winners

### First place: CrossBeam

Mike Brown built CrossBeam to help builders and municipalities address California housing permit delays. The tool parses blueprints and correction letters, assigns subagents to corrections, and generates approval action plans.

### Second place: Elisa

Jon McBee built Elisa, a block-based visual IDE for younger learners. Users assemble specs visually while agents generate code behind the scenes. His daughter used it for a microcontroller project.

### Third place: PostVisit.ai

Cardiologist Michal Nedoszytko built PostVisit.ai to explain diagnoses, analyze visit notes and AI-scribe transcripts, surface clinical evidence, and help patients understand care after an appointment.

### Keep Thinking Prize: TARA

Kyeyune Kazibwe built TARA to turn road dashcam footage into infrastructure investment appraisals, including condition assessment, economic analysis, and equity scoring.

### Special Prize - Creative Exploration: Conductr

Asep Bagja Priandana built Conductr, a browser-based MIDI instrument that uses Claude to generate drums, bass, melody, and harmony in real time based on a live performance.
