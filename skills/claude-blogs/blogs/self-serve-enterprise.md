# Claude Enterprise, now available self-serve

**Source:** https://claude.com/blog/self-serve-enterprise  
**Published:** February 12, 2026  
**Categories:** Enterprise AI, Claude Enterprise, Business

## Overview

Anthropic has made Claude Enterprise available for self-serve purchase directly through its website. Organizations can now set up workspaces, configure SSO, and invite team members within minutes.

## What is Claude Enterprise?

Claude Enterprise provides organizational access to Claude, Claude Code, and Cowork alongside enterprise-grade security features. Teams can:
- Work with large codebases and extensive document collections
- Delegate coding tasks from web and terminal interfaces
- Manage complex multi-step projects with Cowork
- Use plugins customized for sales, finance, legal, and marketing

## Integration Capabilities

Integrates with Microsoft 365, Slack, and other platforms through connectors. Built-in chat sidebars enable direct collaboration within Excel and PowerPoint.

## Security and Administration

- Single sign-on (SSO) and domain capture
- SCIM-based provisioning automation
- Audit logs and Compliance API
- Custom data retention policies
- Usage analytics tracking adoption and spending
- Anthropic does not use Claude Enterprise content for model training by default

## Pricing Model

Seat-plus-usage model. Organizations pay only for what they use at API rates. Administrators can establish spend caps at organizational and individual user levels.

## Getting Started

Begin at claude.ai/create/enterprise. Contact sales for custom terms, tiered usage incentives, HIPAA readiness, or dedicated support.
