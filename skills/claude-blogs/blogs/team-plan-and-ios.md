# Introducing the Claude Team plan and iOS app

**Source:** https://claude.com/blog/team-plan-and-ios  
**Published:** May 1, 2024  
**Categories:** Product announcements, Claude apps

## Overview

Anthropic launched the Claude Team plan and a free iOS application.

## Team Plan

**Pricing**: $30/user/month with a 5-seat minimum.

**Key capabilities**:
- Enhanced usage allowances for increased chat capacity
- Full Claude 3 model access (Opus, Sonnet, Haiku)
- 200K context window for lengthy documents and complex multi-step discussions
- Administrative tools for user management and streamlined billing
- All Pro features (priority access during peak times, early feature releases)

**Upcoming**: Citation verification, data repository integrations, document collaboration capabilities.

## iOS App

Free across all subscription tiers.

**Features**:
- Cross-device synchronization of chat histories
- Vision capabilities for photo analysis and file uploads
- Mobile-optimized functionality for brainstorming and real-time image analysis

Available on the Apple App Store.
