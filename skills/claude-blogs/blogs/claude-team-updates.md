# Updates to Claude Team

**Source:** https://claude.com/blog/claude-team-updates  
**Published:** January 28, 2026  
**Categories:** Product announcements, Claude apps, Work

## Overview

Anthropic has made the Claude Team plan more accessible by lowering prices and introducing annual discounts.

## Pricing Changes

- **Standard seats:** Now $20/month (annual) or $25/month (monthly)
- **Premium seats:** Now $100/month (annual) or $125/month (monthly)

Every Team seat provides more usage than the Pro plan. Premium seats deliver five times the usage of standard seats.

## Key Features

- Shared workspace for teammates to collaborate on projects and access internal knowledge
- Claude Code comes standard with every seat
- Centralized admin controls and billing
- User data remains private by default—Claude doesn't train on team content

Available at claude.com/team.
