# Discover tools that work with Claude

**Source:** https://claude.com/blog/connectors-directory  
**Published:** July 14, 2025  
**Categories:** Product announcements, Claude apps

## Overview

Anthropic launched a directory of tools that integrate with Claude, enabling users to connect third-party applications with a single click—available at claude.ai/directory.

## The Value of Connected Context

Connected tools eliminate the friction of repeatedly explaining project details. Instead of describing everything before Claude can assist, users work directly with their existing information sources:
- "Generate release notes from Linear tickets"
- "Create social media content from Canva briefs"

## Available Connectors

**Remote services**: Notion, Canva, Stripe, and more (requires paid plan)  
**Desktop applications**: Figma, Socket, Prisma (via Claude Desktop app)

## Practical Applications

- Convert discussion threads into organized Notion roadmaps
- Generate design assets from creative briefs in Canva
- Convert Figma designs into production code
- Access Stripe payment and customer data

## Access

Available at claude.ai/directory for all Claude users on web and desktop. Users authenticate with their chosen service.

**Note**: Local desktop extensions require the Claude Desktop app; remote service connectors require a paid plan subscription.
