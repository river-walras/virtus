# What is Model Context Protocol? Connect AI to your world

**Source:** https://claude.com/blog/what-is-model-context-protocol  
**Published:** October 31, 2025  
**Categories:** Agents, Claude apps, Productivity

## Overview

The Model Context Protocol (MCP) is an open standard enabling large language models to communicate with external systems. Think of MCP as **USB-C for LLMs**—providing universal connectivity where previously each application required custom integrations.

## What is MCP?

Created at Anthropic by David Sorria Para and Justin Spahr-Summers. Open-sourced in November 2024. Functions through two components:
- **MCP Clients**: Used by AI assistants
- **MCP Servers**: Built by applications and developers

## Key Benefits

**Universal compatibility**: AI assistants instantly access thousands of MCP-compatible applications; tools reach every compatible AI simultaneously.

**Open ecosystem**: Anyone can build and share MCP servers.

**Foundation for agents**: Enables AI agents to access multiple services for genuine end-to-end task automation.

## Who It's For

- **Developers**: One standard for multiple integrations
- **Enterprises**: Secure, scalable AI connectivity across organizations
- **Consumers**: Seamless connection between AI assistants and work tools
