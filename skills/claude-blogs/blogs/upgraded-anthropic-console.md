# Get to production faster with the upgraded Anthropic Console

**Source:** https://claude.com/blog/upgraded-anthropic-console  
**Published:** March 6, 2025  
**Categories:** Product announcements

## Overview

The redesigned Anthropic Console is now a centralized hub for building, testing, and iterating on AI deployments with Claude, with new collaborative and optimization features.

## Collaborative Development

Share prompts with teammates directly in the Console—no more copy-pasting between documents. Centralized prompt libraries let teams develop, refine, and standardize prompts across the organization.

## Four Core Prompt Capabilities

**Write Prompts**: Workbench provides an interactive environment for structuring prompts, incorporating examples, and integrating external tools while testing API calls.

**Automatic Generation**: Describe your objectives and Claude generates prompts using chain-of-thought reasoning.

**Evaluation**: Test prompts against real-world scenarios with automatic test case generation and side-by-side comparisons graded on a 5-point scale.

**Refinement**: Automatically improves prompts using advanced engineering techniques—especially helpful when adapting prompts from other AI models.

## Extended Thinking Support

Claude 3.7 Sonnet integration lets users control thinking token budgets, optimizing for either immediate answers or detailed step-by-step reasoning visible to users.

## Availability

Available to all users at console.anthropic.com.
