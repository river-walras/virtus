# Claude Cowork: Enterprise-Grade AI for Knowledge Work

**Source:** https://claude.com/blog/cowork-research-preview  
**Categories:** Product announcements, Claude Enterprise

## Overview

Claude Cowork is Anthropic's agentic AI system for delegating knowledge work tasks. Unlike Chat (conversational responses), Cowork executes multi-step tasks independently—organizing files, building spreadsheets, preparing reports, and analyzing notes.

## Key Capabilities

### Task Execution

Cowork completes work autonomously:
- File organization and renaming
- Spreadsheet creation and data extraction
- Report generation from scattered materials
- Scheduled recurring tasks

Accepts task descriptions and delivers finished deliverables rather than step-by-step instructions.

### Computer Use and Integration

- **Direct connectors**: Slack, Chrome, and company systems
- **Computer use**: Interact with applications when integrations aren't available
- **Local file access**: Folders you explicitly authorize

"Claude works through each step, looping you in before anything significant."

### Scheduled Automation

Establish recurring workflows—daily briefings, weekly metrics pulls, monthly report updates—that run without manual intervention.

## Enterprise Features

- Manage feature access across teams
- Control spending and usage tracking
- Role-based access controls
- Security permissions configuration

## Safety and Control

User approval gates require confirmation before financial, personal, or business-critical actions. Cowork displays its planned approach and awaits authorization. Administrators can toggle Cowork access organization-wide.

## Pricing

- Pro: $17–$20/month (Cowork included)
- Max: $100–$200/month (enhanced allocation)
- Team: $20/seat/month
- Enterprise: Custom pricing

## Current Limitations

Not suitable for HIPAA, FedRAMP, or regulated financial workloads. Computer use remains in research preview.
