# Remote MCP Support in Claude Code

**Source:** https://claude.com/blog/claude-code-remote-mcp  
**Published:** June 18, 2025  
**Categories:** Product announcements, Claude Platform, Coding

## Overview

Claude Code now supports remote MCP servers, enabling developers to connect third-party tools and data sources securely without managing local infrastructure.

## Integration Capabilities

Claude Code can access tools and resources exposed by MCP servers, pulling context from development tools, project management systems, and knowledge bases.

**Key integrations**:
- **Sentry**: Access errors and issues directly within Claude Code for debugging without switching contexts
- **Linear**: Work with active projects and issue details during the development workflow

Linear's Head of Engineering: "Stay in flow when moving between planning, writing code, and managing issues. Fewer tabs, less copy-paste. Better software, faster."

## Key Features

**Low Maintenance**: Vendors handle updates, scaling, and availability. Users simply add the vendor's URL to Claude Code—no manual setup required.

**Native OAuth Support**: Secure authentication to existing accounts eliminates API key management and credential storage.

## Getting Started

Available now in Claude Code. Add a vendor's URL in Claude Code configuration. See the documentation and the MCP directory at anthropic.com/partners/mcp for recommended servers.
