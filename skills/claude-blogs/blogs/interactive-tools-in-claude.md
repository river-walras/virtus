# Your Favorite Work Tools Are Now Interactive Connectors Inside Claude

**Source:** https://claude.com/blog/interactive-tools-in-claude  
**Published:** January 26, 2026  
**Categories:** Product announcements, Claude apps, Content Creation

## Overview

Anthropic has launched MCP Apps, bringing interactive connectors directly into Claude conversations. Users can now access and control productivity tools without tab-switching.

## Available Connectors

- **Amplitude** – Create and explore analytics charts interactively
- **Asana** – Convert conversations into projects, tasks, and timelines
- **Box** – Search files and extract document insights
- **Canva** – Design presentation decks with real-time customization
- **Clay** – Research companies and draft personalized outreach
- **Figma** – Generate flowcharts and diagrams in FigJam
- **Hex** – Query data with interactive charts and citations
- **monday.com** – Manage projects and assign tasks
- **Slack** – Search conversations and draft formatted messages
- Salesforce integration coming soon

## Technical Foundation

MCP Apps builds on the **Model Context Protocol (MCP)**, an open standard for connecting tools to AI applications. This extension enables developers to create rich user interfaces atop MCP servers across any supporting AI product.

## Availability

Available across Claude on mobile, web, and desktop for all subscription tiers. Visit claude.ai/directory.
