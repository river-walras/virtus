# Claude Code and new admin controls for business plans

**Source:** https://claude.com/blog/claude-code-and-new-admin-controls-for-business-plans  
**Published:** August 20, 2025  
**Categories:** Product announcements

## Overview

Enterprise and Team customers can now upgrade to premium seats combining Claude and Claude Code in a single subscription, with new admin controls for visibility and governance.

## Premium Seats

Administrators flexibly assign standard or premium seats. Premium seats grant access to both Claude and Claude Code:
- Research in Claude conversations → implement in Claude Code terminal
- Explore architectural trade-offs in chat → build chosen solution with Claude Code

Early adopters: "Since bundling Claude for Enterprise with Claude Code, we've rolled it out to hundreds of developers, and it has quickly become our go-to pair programmer."

## Flexible Pricing

Claude seats include sufficient usage for a typical workday. Admins can enable extra usage at standard API rates. Administrators control maximum spending per user.

## Admin Controls

- **Self-serve seat management**: Purchase, allocate, and provision users through admin panel
- **Granular spend controls**: Set spending limits at organization and individual user levels
- **Usage analytics**: View Claude Code metrics (lines of code accepted, suggestion accept rates)
- **Managed policy settings**: Deploy settings across Claude Code users for tool permissions and file access restrictions

## Compliance API

New Compliance API lets enterprise organizations meet regulatory requirements programmatically:
- Access usage data and customer content without manual exports
- Integrate Claude data into compliance dashboards
- Automatically flag potential issues
- Manage data retention through selective deletion
