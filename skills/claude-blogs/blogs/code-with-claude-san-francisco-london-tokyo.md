# Code with Claude comes to San Francisco, London, and Tokyo

**Source:** https://claude.com/blog/code-with-claude-san-francisco-london-tokyo  
**Published:** March 18, 2026  
**Categories:** Claude Code

## Overview

Anthropic is expanding its developer conference, Code with Claude, to three cities this spring. The event features technical sessions, live demonstrations of new capabilities, and one-on-one office hours with Anthropic engineers.

## Event Details

The full-day conference is designed for developers and founders working with Claude. Expanded technical sessions and talks from Anthropic engineers alongside developers at startups and enterprises shipping Claude-based solutions.

**Locations and Dates:**
- **San Francisco:** May 6, 2026
- **London:** May 19, 2026
- **Tokyo:** June 10, 2026

In-person attendance is limited across all three cities. Livestream access and session recordings are available to all interested viewers.
