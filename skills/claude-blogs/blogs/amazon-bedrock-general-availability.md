# Claude on Amazon Bedrock now available to every AWS customer

**Source:** https://claude.com/blog/amazon-bedrock-general-availability  
**Published:** September 28, 2023  
**Categories:** Product announcements

## Overview

Claude is now generally available on Amazon Bedrock—the fully managed AWS service providing secure cloud access to foundation models for building generative AI applications.

## Agents for Amazon Bedrock

Claude helped develop Agents for Amazon Bedrock (preview). These agents can:
- Orchestrate and perform API calls through AWS Lambda functions
- Understand requests and decompose complex tasks
- Conduct conversations and retrieve information
- Execute actions to fulfill user needs

Example: A chat assistant that goes beyond inventory queries to help customers update orders and access manuals.

## Upcoming Features

Anthropic plans to offer secure customization and fine-tuning of Claude models through Bedrock, optimizing performance with customer data while limiting harmful outputs.

## Enterprise Adoption

Since April announcement: adoption by LexisNexis, Lonely Planet, Ricoh USA.

**Bridgewater Associates**: Developed an AI-enabled Investment Analyst Assistant. "Claude is the highly capable model behind our Investment Analyst Assistant, able to take basic instructions, generate Python code, work through errors, and output charts."
