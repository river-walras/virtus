# Workspaces in the Anthropic API Console

**Source:** https://claude.com/blog/workspaces  
**Published:** September 10, 2024  
**Categories:** Product announcements, Claude Platform

## Overview

Workspaces in the API Console help developers manage multiple Claude deployments more efficiently—unique environments for structuring resources, refining access controls, and establishing custom spend and rate limits.

## Key Capabilities

- **Granular spend limits**: Monthly spend limits per workspace for fine-grained cost control
- **Group related resources**: Organize API keys, usage data, and settings into logical groups aligned with projects or environments
- **Manage rate limits**: Adjust rate limits independently per workspace while respecting organization-wide limits
- **Streamline access control**: Assign user permissions at the workspace level for enhanced security
- **Monitor usage**: Track API usage and costs broken down by workspace for optimized resource allocation

## Use Cases

Developers deploying Claude across multiple environments (development, staging, production) and different use cases can use Workspaces as an abstraction layer for the overall organization and individual API keys.

## Getting Started

Available to all Anthropic API Console users. Create workspace-scoped API keys at console.anthropic.com/settings/workspaces.
