# How a non-technical project manager built and shipped a stress management app with Claude Code in six weeks

**Source:** https://claude.com/blog/how-a-non-technical-project-manager-built-and-shipped-a-stress-management-app-with-claude-code-in-six-weeks  
**Published:** May 1, 2026  
**Categories:** Claude Code

## Overview

This founder story profiles Kostiantyn Vlasenko, a project manager with no prior coding experience who built and shipped Respiro, an iOS stress management app, with Claude Code.

## Respiro

Respiro detects stress signals from personal devices and intervenes with guided breathing exercises when the user appears stressed. Vlasenko initially built the app during the Built With Opus 4.6 Claude Code Hackathon and later shipped it on the App Store.

## Claude Code Workflow

Vlasenko treated Claude Code agents like a team. He used Claude to research Apple APIs, design a multi-agent architecture, and coordinate specialized agents such as a TCA architect, Swift developer, Metal specialist, and code reviewer.

When his React Native MVP became hard to test without Android hardware, Claude Code helped rewrite the app in Swift in a few hours.

## Shipping and Growth

Claude helped guide App Store submission, third-party integrations such as Sentry and Amplitude, analytics funnels, content creation, and growth strategy. Vlasenko also used Claude's vision capabilities for screenshots and setup flows.

## Organizational Impact

Vlasenko brought the workflow back to Mythical Games, where he became an internal advocate for Claude Code workflows and helped a small internal team ship work through Claude.
