# Claude now creates interactive charts, diagrams and visualizations

**Source:** https://claude.com/blog/claude-builds-visuals  
**Published:** March 12, 2026  
**Categories:** Product announcements, Claude apps, Education

Claude has introduced a new capability enabling it to build interactive charts, diagrams, and visualizations directly within conversations. Unlike the permanent artifacts feature, these visuals appear inline and adapt as discussions evolve.

## Key Features

- Claude automatically decides when a visual would enhance understanding, or users can explicitly request visualizations
- Interactive compound interest curves and clickable periodic table visualizations
- Complements specialized layouts for recipes, weather visualizations
- Users can interact with integrated apps including Figma, Canva, and Slack directly within conversations

## Availability

Launches in beta across all Claude subscription tiers. The visualizations are designed as temporary, conversational aids rather than polished deliverables.
