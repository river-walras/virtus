# Claude 2 on Amazon Bedrock

**Source:** https://claude.com/blog/claude-2-amazon-bedrock  
**Published:** August 23, 2023  
**Categories:** Product announcements, Claude Platform

## Overview

Claude 2 is available on Amazon Bedrock, making it easier for businesses to build and scale generative AI applications with enhanced security and reliability built on Constitutional AI principles.

## Enterprise Implementations

**LexisNexis Legal Services**: Selected Claude 2 for AI-assisted legal capabilities. Valued technical analysis quality, extended context windows for document processing, and rapid response times.

**Lonely Planet Travel**: Integrated Claude 2 to help customers access decades of travel content. Scalable, secure generative AI integration for travel planning assistance.

**Ricoh USA Operations**: Deployed Claude 2 to generate high-quality training datasets while preserving data integrity. Meets HIPAA and SOC II compliance standards.
