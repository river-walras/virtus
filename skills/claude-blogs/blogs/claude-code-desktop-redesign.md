# Redesigning Claude Code on desktop for parallel agents

**Source:** https://claude.com/blog/claude-code-desktop-redesign  
**Published:** April 14, 2026  
**Categories:** Claude Code, Product announcements

## Overview

Anthropic released a redesigned Claude Code desktop app built around running and managing multiple coding-agent sessions at once.

## New Desktop Experience

### Parallel sessions

The app adds a sidebar for active and recent sessions, with filtering by status, project, or environment. Sessions can be grouped by project and archived automatically when PRs merge or close.

### Side chats

Users can branch into a side chat with context from the main thread without feeding that side conversation back into the active task.

### Built-in review and shipping tools

The redesign adds an integrated terminal, in-app file editor, faster diff viewer, expanded preview support for HTML and PDFs, and drag-and-drop pane layouts.

### Stack support

The desktop app reaches parity with Claude Code CLI plugins. Users can still run sessions locally or in the cloud, and SSH support extends to Mac alongside Linux.

### Customization

Verbose, Normal, and Summary view modes control how much of Claude's tool activity is visible. New keyboard shortcuts cover session switching, spawning, and navigation.

## Availability

The redesigned desktop app is available for Claude Code users on Pro, Max, Team, and Enterprise plans, and through the Claude API.
