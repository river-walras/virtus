# Advancing Claude for Excel and PowerPoint

**Source:** https://claude.com/blog/claude-excel-powerpoint-updates  
**Published:** March 11, 2026  
**Categories:** Enterprise AI, Claude Enterprise

## Overview

Claude for Excel and PowerPoint now enable seamless collaboration across multiple open files within a single conversation, allowing users to work more efficiently without repeatedly re-explaining datasets.

## Key Features

### Unified Context Across Applications
Claude maintains full context when moving between Excel and PowerPoint files. A financial analyst can pull comparable company financials from an open workbook and build analysis and presentations without context loss.

### Skills for Workflow Automation

**Excel Skills:**
- Formula error auditing and balance-sheet verification
- Financial modeling (LBO, DCF, 3-statement templates)
- Comparable company analysis
- Data cleaning across ranges, sheets, or entire files

**PowerPoint Skills:**
- Competitive landscape deck development
- Updating presentations with new data
- Investment banking deck review for consistency

Skills sync automatically from Claude's desktop or web app to the add-ins without additional configuration.

### Instructions for Persistent Preferences
Users can establish app-level settings that apply automatically, such as number formatting standards.

## Deployment Options
- Direct Claude accounts
- Amazon Bedrock
- Google Cloud's Vertex AI
- Microsoft Foundry
- Claude powers Agent Mode natively within Excel for Microsoft 365 Copilot customers

## Availability
All Mac and Windows users on paid plans can access these features in beta.
