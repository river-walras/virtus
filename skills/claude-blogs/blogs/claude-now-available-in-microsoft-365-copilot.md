# Claude is now available in Microsoft 365 Copilot

**Source:** https://claude.com/blog/claude-now-available-in-microsoft-365-copilot  
**Published:** September 24, 2025  
**Categories:** Product announcements, Claude apps

## Overview

Claude models are now integrated into Microsoft 365 Copilot, expanding access across millions of enterprise users. Organizations can select Claude Sonnet 4 and Opus 4.1 as model options.

## Integration Points

### Researcher Agent
Supports **Claude Opus 4.1** for advanced reasoning. Use for complex research spanning web sources, third-party data, and organizational content (emails, chats, meetings, files). Choose your preferred AI for multistep analysis and report generation.

### Copilot Studio
Both **Claude Sonnet 4** and **Opus 4.1** available. Organizations build, customize, and deploy enterprise-grade agents using Anthropic models. Multiagent system support allows specialized task assignment based on model capabilities.

## Getting Started

Rollout begins through Microsoft's Frontier Program for licensed Microsoft 365 Copilot customers who opt in. Administrators enable Anthropic model access through the Microsoft 365 admin center.

**Note**: Anthropic models operate outside Microsoft-managed infrastructure and remain subject to Anthropic's Terms and Conditions.
