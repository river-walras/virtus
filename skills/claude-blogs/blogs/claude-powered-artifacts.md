# Build and share AI-powered apps with Claude

**Source:** https://claude.com/blog/claude-powered-artifacts  
**Published:** July 25, 2025  
**Categories:** Product announcements, Claude apps, Work

## Overview

Developers can now create, host, and share interactive AI-powered applications directly within Claude—eliminating traditional deployment complexity and scaling costs.

## How It Works

Artifacts interact with Claude through an API, becoming functional AI applications. When users access a Claude-powered app:
- Authentication uses existing Claude accounts
- API usage counts against the **user's** subscription, not the creator's
- Creators incur no costs for user activity
- No API key management required

Claude generates executable code that orchestrates sophisticated AI functionality—creators can view, modify, and distribute freely.

## Early Use Cases

- AI-powered games with adaptive NPCs and conversation memory
- Personalized learning tools with skill-based tutoring
- Data analysis platforms accepting CSV uploads for natural language queries
- Writing assistants for scripts and technical documentation
- Complex agent workflows orchestrating multiple Claude API calls

## Getting Started

Describe your desired application in Claude → Claude generates the code → Share instantly via link (no deployment required). Claude handles prompt engineering, error handling, and orchestration.

## Current Capabilities and Limitations

**Available**: Claude API integration within artifacts, file processing, React-based rich UI creation, artifact forking and customization.

**Not yet available**: External API calls, persistent storage, streaming API.

## Availability

Beta for Free, Pro, and Max plan users; available on Team and Enterprise as of July 31, 2025.
