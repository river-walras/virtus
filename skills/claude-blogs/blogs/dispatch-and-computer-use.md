# Put Claude to work on your computer

**Source:** https://claude.com/blog/dispatch-and-computer-use  
**Published:** March 23, 2026  
**Categories:** Product announcements, Claude apps, Coding

Claude can now use your computer to complete tasks directly. In Claude Cowork and Claude Code, the AI can control your mouse, keyboard, and browser to navigate applications, open files, and run developer tools—all with explicit permission.

## How Computer Use Works

Claude prioritizes direct tool integrations first (connectors for Slack or Google Calendar). When those aren't available, it can interact with your screen directly. The system includes safeguards against prompt injection and automatically scans for suspicious activity. Users retain full control and can stop Claude anytime.

The capability remains in early research preview. Claude can make mistakes, and complex tasks may require multiple attempts. Some apps are restricted by default for security reasons.

## Dispatch: Task Assignment from Anywhere

Dispatch is a new feature allowing continuous conversations with Claude across devices. Users can assign tasks via their phone, attend to other matters, then review completed work on their desktop.

Examples:
- Claude checking emails automatically
- Pulling weekly metrics
- Creating morning briefings
- Making IDE changes with automated testing and pull requests
- Managing ongoing projects

## Getting Started

Launches in research preview for Claude Pro and Max subscribers on macOS and Windows. Enable computer use in desktop app settings, pair with the mobile app, and assign tasks from your phone.
