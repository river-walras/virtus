# How enterprises are building AI agents in 2026

**Source:** https://claude.com/blog/how-enterprises-are-building-ai-agents-in-2026  
**Published:** December 9, 2025  
**Categories:** Enterprise AI

## Overview

A survey of over 500 technical leaders reveals enterprises are rapidly deploying AI agents across operations, moving beyond simple task automation to tackle complex, multi-step workflows.

## Key Findings

- 57% deploy agents for multi-stage workflows; 16% run cross-functional processes
- 81% plan to tackle more complex use cases in 2026
- Nearly 90% use AI to assist with development; 86% deploy agents for production code
- Time savings across: planning (58%), code generation (59%), documentation (59%), testing (59%)
- 80% of organizations report measurable economic returns on AI agent investments

## Real-World Applications

- **Thomson Reuters**: CoCounsel enables lawyers to access 150 years of case law in minutes
- **eSentire**: Compressed threat analysis from 5 hours to 7 minutes with 95% accuracy
- **Doctolib**: Shipped features 40% faster after deploying Claude Code
- **L'Oréal**: 99.9% accuracy on conversational analytics for 44,000 monthly users

## Implementation Challenges

1. Integration with existing systems (46%)
2. Data access and quality (42%)
3. Change management (39%)
