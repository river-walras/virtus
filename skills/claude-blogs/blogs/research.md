# Claude Takes Research to New Places

**Source:** https://claude.com/blog/research  
**Published:** April 15, 2025  
**Categories:** Product announcements

## Overview

Anthropic unveiled two significant capabilities: the Research feature for agentic web search and Google Workspace integration.

## Research Capability

Claude conducts agentic web searches—multiple searches that build on each other, exploring questions from multiple angles. Delivers comprehensive answers with citations in minutes.

Supports professional workflows across domains by combining research speed with quality output.

## Google Workspace Integration

Claude connects with Gmail, Calendar, and Google Docs to search emails, review documents, and access calendar information—no manual file uploads or repeated context-sharing needed.

**Use Case Applications**:
- Marketing: competitive intelligence + product specifications
- Sales: client briefings from correspondence history + prospect research
- Engineering: design documents + external API documentation
- Students: learning materials + academic research
- Parents: family schedules + local events

## Google Docs Cataloging

Enterprise administrators can enable document cataloging to improve retrieval accuracy across organizational knowledge bases using secure retrieval-augmented generation techniques.

## Availability

Research in early beta for Max, Team, and Enterprise plans in the United States, Japan, and Brazil. Google Workspace integration in beta for all paid users.
