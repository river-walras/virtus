# Improve your prompts in the developer console

**Source:** https://claude.com/blog/prompt-improver  
**Published:** October 14, 2024  
**Categories:** Product announcements, Claude Platform

## Overview

The Anthropic Console now includes prompt improvement capabilities—automatically refining prompts using advanced engineering techniques.

## Prompt Improver Methods

1. **Chain-of-thought reasoning**: Adds dedicated sections for systematic problem-solving before responding
2. **Example standardization**: Converts examples into consistent XML formatting
3. **Example enrichment**: Augments examples with reasoning aligned to the structured prompt
4. **Rewriting**: Clarifies structure and corrects grammatical issues
5. **Prefill addition**: Directs actions and enforces output formats

**Measured results**: 30% accuracy increase for multilabel classification tests; 100% word count adherence for summarization tasks.

## Multi-Shot Example Management

Developers manage examples in structured formats within the Workbench—add input/output pairs, edit existing examples, or auto-generate synthetic examples. Improves accuracy, consistency, and performance on complex tasks.

## Prompt Evaluation

Optional "ideal output" column in the Evaluations tab for grading model outputs on a 5-point scale—useful for benchmarking and improvement.

## Customer Spotlight

**Kapa.ai**: Used the prompt improver to migrate workflows to Claude 3.5 Sonnet—"streamlined our migration to Claude 3.5 Sonnet and enabled us to get to production faster."

## Getting Started

Available to all users in the Anthropic Console at console.anthropic.com.
