# Claude for Enterprise

**Source:** https://claude.com/blog/claude-for-enterprise  
**Published:** September 10, 2024  
**Categories:** Product announcements

## Overview

The Claude Enterprise plan enables organizations to collaborate with Claude securely while leveraging internal knowledge at scale.

## Key Features

- **500K context window**: Equivalent to hundreds of sales transcripts, dozens of 100+ page documents, or medium-sized codebases
- **Native GitHub integration** (beta): Engineering teams sync repositories with Claude for feature iteration, debugging, and onboarding
- **Projects and Artifacts**: Combined with expanded context for end-to-end solutions from concept to completion

## Enterprise-Grade Security Controls

- Single sign-on (SSO) and domain capture
- Role-based access with fine-grained permissions
- Audit logs for security and compliance monitoring
- SCIM for automated user provisioning

## Customer Examples

- **GitLab**: Brainstorming, process optimization, content creation, code writing — "Claude offers our team members a tool that feels like an extension of their work and expertise."
- **Midjourney**: Various workflows from content creation to code writing
