# Claude can now create and edit files

**Source:** https://claude.com/blog/create-files  
**Published:** September 9, 2025  
**Categories:** Product announcements, Claude apps, Claude Enterprise

## Overview

Claude can now create and edit Excel spreadsheets, documents, PowerPoint presentations, and PDFs directly within Claude.ai and the desktop app. Users input requirements, share data, and download finalized files.

## Key Capabilities

- **Data transformation**: Convert raw datasets into polished outputs with statistical analysis, visualizations, and written summaries
- **Spreadsheet creation**: Build financial models, project trackers, budget templates, and dashboards with formulas and multiple sheets
- **Cross-format conversion**: Transform PDFs into presentations, meeting notes into formatted documents, invoices into organized spreadsheets

## How It Works

Claude accesses "a private computer environment where it can write code and run programs" to generate files and analyses—shifting from providing guidance to active collaboration.

## Availability

Launched as preview for Max, Team, and Enterprise plan subscribers; Pro coming shortly. Generally available for paid plans as of October 21, 2025, with network and egress controls.

## Getting Started

Enable "Upgraded file creation and analysis" in Settings > Features > Experimental.

**Security note**: This feature grants Claude internet access. Monitor chats closely when using it.
