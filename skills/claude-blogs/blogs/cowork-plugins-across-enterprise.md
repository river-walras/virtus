# Cowork and Plugins for Teams Across the Enterprise

**Source:** https://claude.com/blog/cowork-plugins-across-enterprise  
**Published:** February 24, 2026  
**Categories:** Agents, Claude Enterprise

## Overview

Anthropic has introduced updates to Cowork and plugins designed to help enterprises customize Claude for organizational workflows. Enables teams to build private plugin marketplaces and simplifies plugin creation.

## Plugin Experience Upgrades

- Administrators can now establish plugins from starter templates or build custom solutions
- A new unified "Customize" menu consolidates plugins, skills, and connectors
- Org-specific marketplaces, private GitHub repositories as plugin sources (private beta)
- Per-user provisioning and automatic installation options
- Slash commands now launch with structured forms
- OpenTelemetry support enables admins to monitor usage, costs, and tool activity

## New Connectors

Google Workspace (Calendar, Drive, Gmail), Docusign, Apollo, Clay, Outreach, Similarweb, MSCI, LegalZoom, FactSet, WordPress, Harvey, Slack, LSEG, S&P Global.

## Pre-Built Plugin Templates

- **HR**: Employee lifecycle management, offer letters, performance reviews
- **Design**: Critique frameworks and accessibility audits
- **Engineering**: Standup summaries and incident coordination
- **Operations**: Documentation, vendor evaluations, runbooks
- **Brand Voice**: Distill brand guidelines from existing materials
- Plus: Financial Analysis, Investment Banking, Equity Research, Private Equity, Wealth Management

## Cross-Application Capabilities

Claude can handle multi-step tasks across Excel and PowerPoint, maintaining context between Office add-ins.

## Availability

User experience updates for plugins available to all Cowork users. Company branding and provisioning controls accessible to Team and Enterprise admins.
