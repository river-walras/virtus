# Claude can now search the web

**Source:** https://claude.com/blog/web-search  
**Published:** March 20, 2025  
**Categories:** Product announcements, Claude Platform

## Overview

Claude now offers web search capabilities across all plans, enabling access to current information and cited responses.

## Key Capabilities

- Accesses real-time data when relevant to conversations
- Provides citations for fact-checking
- Processes and delivers relevant sources in conversational language (not raw search results)
- Expands knowledge base with contemporary insights

## Use Cases

- **Sales teams**: Industry trends and prospect pain points for account planning
- **Financial analysts**: Current market data and earnings reports for investment decisions
- **Researchers**: Emerging trends and literature gaps for grant proposals and reviews
- **Shoppers**: Compare product features, pricing, and reviews across multiple sources

## Availability

Initially available for paid U.S. users in feature preview. May 2025: expanded globally to all paid plans. Subsequently extended to free plan users and additional countries.

Enable web search in profile settings; interact with Claude 3.7 Sonnet to activate when relevant.
