# Claude Security is now in public beta

**Source:** https://claude.com/blog/claude-security-public-beta  
**Published:** April 30, 2026  
**Categories:** Product announcements, Claude Security

## Overview

Claude Security became available in public beta for Claude Enterprise customers. It lets teams scan code for vulnerabilities and generate proposed fixes using Claude Opus 4.7.

## How Claude Security Works

Users select a repository, directory, or branch from Claude.ai or `claude.ai/security`, then start a scan. Claude reasons across files and modules, traces data flows, and looks for context-dependent vulnerabilities rather than only pattern-matching known issues.

For each finding, Claude provides:

- Confidence that the vulnerability is real
- Severity and likely impact
- Reproduction context
- Targeted patch instructions
- A path to work through the fix in Claude Code on the Web

## Preview Learnings

Anthropic says early enterprise use emphasized three needs:

- High-confidence findings with fewer false positives
- Shorter time from scan to applied fix
- Ongoing scheduled coverage rather than one-off audits

New features include scheduled scans, directory-level targeting, dismissal reasons, CSV/Markdown exports, and webhooks to Slack, Jira, or other systems.
