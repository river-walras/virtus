# Claude Android App

**Source:** https://claude.com/blog/android-app  
**Published:** July 16, 2024  
**Categories:** Product announcements

## Overview

Claude launched an Android application, extending availability across mobile and desktop platforms. Provides access to Claude 3.5 Sonnet and is compatible with all subscription tiers.

## Key Features

- **Cross-platform continuity**: Start conversations on one device, resume seamlessly across web, iOS, and Android
- **Visual analysis**: Capture images and upload files for real-time analysis
- **Language support**: Built-in translation features for multiple languages
- **Complex problem-solving**: Contract review, market analysis, strategic planning

## Use Cases

- Professional work during meetings
- Menu translation while traveling
- Creative brainstorming during shopping
- Speech composition while awaiting travel

## Access

Available on Google Play across all subscription plans (including Pro and Team).
