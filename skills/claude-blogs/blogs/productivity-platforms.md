# Claude and your productivity platforms

**Source:** https://claude.com/blog/productivity-platforms  
**Published:** October 16, 2025  
**Categories:** Product announcements, Claude Enterprise, Productivity

## Overview

Claude integrates with Microsoft 365 and offers enterprise search across connected tools, bringing context from documents, emails, and calendars directly into conversations.

## Microsoft 365 Connector

Connect Claude with Microsoft 365 via the MCP connector. Claude gains context to reason through complex problems and take action faster.

Works with:
- **Microsoft SharePoint and OneDrive**: Search and analyze documents across sites and libraries without manually uploading files
- **Microsoft Outlook**: Access email threads, analyze communication patterns, extract insights about project status and client feedback
- **Microsoft Teams**: Search chat conversations, channel discussions, and meeting summaries to surface decisions and track project updates

## Enterprise Search

A dedicated shared project personalized with your company name makes accessing company knowledge easier. Includes custom prompts for searching across connected tools plus simple setup steps. Everyone gets automatic access once an admin sets it up.

When you ask Claude a question, it searches across all connected data sources at once—for example, asking about remote work policy retrieves information from HR documents, email discussions, and team guidelines in one report.

**Best for**: Onboarding new team members, answering strategic questions, identifying internal experts.

## Getting Started

- Microsoft 365 connector: available for all Claude Team and Enterprise customers. Admins enable first, then users authenticate.
- Enterprise search: enabled for all Team and Enterprise organizations. Admins customize the project and curate data sources.
