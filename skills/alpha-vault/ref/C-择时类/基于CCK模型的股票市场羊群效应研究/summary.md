# 概述

羊群效应通常来说，是由于个别股票的暴涨或暴跌引起其他相关股票收 益率联动，致某类股票暴涨或暴跌。因此，通俗来说，羊群效应刻画的
是个股之间联动性的变化，并于此对趋势强度进行判断。
